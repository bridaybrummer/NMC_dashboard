SA NMC Dashboard

This dashboard aims to make NMC data accessible and actionable for public health officials across South Africa. It provides timely insights into disease trends, hotspots, and forecasts to support effective decision-making and resource allocation.

In addition, all coding and analysis will be open sources so that an epidemiologist can reproduce it locally, but also improve the code.

Currently, only the by-province section is functional as a prototype to estbalished the structure of the dashboard/website.
